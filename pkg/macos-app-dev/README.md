# `macos-app-dev`

Installs macOS GUI applications for development workflows.

## Installs

| App                     | macOS  | Debian |
| ----------------------- | ------ | ------ |
| Hex Fiend               | `brew` | —      |
| Jupyter Notebook Viewer | `brew` | —      |
| Kaleidoscope            | `brew` | —      |
| Postman                 | `brew` | —      |

> Also brings in `macos-iterm`, `macos-sublime`, `macos-terminal`, `macos-vscode`, and `macos-xcode` as make dependencies.

## Configures

_(no configuration — installs only)_

## Notes

- All four apps are installed as Homebrew casks via `app-dev.Brewfile`.
- Kaleidoscope is a paid app; a license is required after install.

## References

- [Hex Fiend](https://hexfiend.com): Fast, open-source hex editor for macOS.
- [Jupyter Notebook Viewer](https://nbviewer.org): Renders `.ipynb` notebooks without a running Jupyter server.
- [Kaleidoscope](https://kaleidoscope.app): File diff and merge tool with support for text, images, and folders.
- [Postman](https://www.postman.com): API client for building, testing, and documenting HTTP requests.
