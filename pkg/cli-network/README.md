# `cli-network`

Network diagnostic and utility CLIs for macOS and Debian/Ubuntu.

## Installs

- Brew: `inetutils`, `iproute2mac`, `netcat`, `nmap` (`pkg/cli-network/cli-network.Brewfile`)
- APT: `iproute2`, `iputils-ping`, `dnsutils`, `netcat-traditional`, `net-tools`, `nmap`, `telnet`, `traceroute` (`pkg/cli-network/cli-network.Aptfile`)
